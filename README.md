# Umbraco v14+ Self-Study Demo

## Introduction
This project documents my journey into **Umbraco v14**, focusing on the new **"Bellissima" Backoffice architecture**.
As a Senior .NET Developer transitioning to the Umbraco ecosystem, I built this demo to demonstrate my quick adaptability and understanding of modern frontend integrations using **Lit** and **TypeScript**.

## Tech Stack
* **Backend:** ASP.NET Core 8.0, C#
* **CMS:** Umbraco v14+
* **Frontend (Backoffice):** TypeScript, Lit (Web Components)
* **Database:** SQLite (for portability) / SQL Server

## Features Implemented (Plan)
* [x] Initial Setup & Configuration.
* [ ] Custom Document Types & Templates.
* [ ] **Custom Property Editor (using TypeScript & Lit)** - *In Progress*.
* [ ] Headless API Endpoint implementation.

## How to Run
1.  Clone this repository.
2.  Run `dotnet run`.
3.  Access `https://localhost:443xx/umbraco` to see the work.
